Generic Strategy Class

Description

This class defines a common interface for generic algorithm strategies (e.g., NMFStrategy-class).

name and name<- gets and sets the name associated with an object. In the case of Strategy objects it is the the name of the algorithm.

Usage

name(object, ...)

S4 (Strategy)
`name`(object, all = FALSE)

name(object, ...) <- value

Arguments

object
an R object with a defined name method
...
extra arguments to allow extension
value
replacement value
all
a logical that indicates if all the names associated with a strategy should be returned (TRUE), or only the first (primary) one (FALSE).

Slots

  1. namecharacter string giving the name of the algorithm

  2. packagename of the package that defined the strategy.

  3. defaultsdefault values for some of the algorithm's arguments.

Methods

  1. namesignature(object = "Strategy"): Returns the name of an algorithm

  2. namesignature(object = "Strategy"): Returns the name of an algorithm

  3. name<-signature(object = "Strategy", value = "character"): Sets the name(s) of an NMF algorithm

  4. name<-signature(object = "Strategy", value = "character"): Sets the name(s) of an NMF algorithm

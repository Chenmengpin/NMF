TRANSFAC Model - Multidimensional Nonnegative Matrix Factorization

Description

This class provides data structure for NMF array models (e.g., TRANSFAC models), where the basis and/or the coefficient data is stored in a multidimensional array.

Methods

  1. .basissignature(object = "NMFarray"): Get the basis matrix in NMF array models

    This function returns slot W of object.

  2. .basis<-signature(object = "NMFarray", value = "matrix"): Replaces a slice of the basis array.

  3. .basis<-signature(object = "NMFarray", value = "array"): Set the basis array in NMF array models

    This function sets slot W of object.

  4. .coefsignature(object = "NMFarray"): Get the mixture coefficient array in standard NMF array models

    This function returns slot H of object.

  5. .coef<-signature(object = "NMFarray", value = "array"): Set the mixture coefficient matrix in standard NMF models

    This function sets slot H of object.

  6. .coef<-signature(object = "NMFarray", value = "matrix"): Replaces a slice of the coefficent array.

  7. fittedsignature(object = "NMFarray"): Computes the target matrix estimate for an NMF array model.

  8. showsignature(object = "NMFarray"): Show method for objects of class NMFarray

Examples



# create a completely empty NMF object
new('NMFarray')

## <Object of class:NMFarray>
## features: 0 
## basis/rank: 0 
## samples: 0 
## levels:  1 | 1

# create a NMF object based on random (compatible) arrays
n <- 4; r <- 3; p <- 5; q <- 2
w <- array(seq(n*r*q), dim = c(n, r, q))
h <- rmatrix(r, p)
new('NMFarray', W=w, H=h)

## <Object of class:NMFarray>
## features: 4 
## basis/rank: 3 
## samples: 5 
## levels:  2 | 1
